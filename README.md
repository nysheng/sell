# sell
微信点餐系统
## 功能点
1 买家端：
> - 关注公众号
> - 微信端打开链接查看商品信息
> - 选择商品，加入购物车
> - 填写信息，提交订单
> - 微信支付
> - 获得微信支付凭证

2 卖家端
> - 扫码登录后端管理系统
> - 接收新订单通知
> - 查看订单列表，处理订单
> - 管理商品信息以及类目信息
> - 退出系统

## 技术点
> - spring boot
> - lombok
> - jpa
> - freemarker模板引擎技术
> - redis
> - mysql5.7.17
> - bootstrap
> - 微信公众号网页授权
> - 微信授权支付
> - 公众号推送模板消息
> - 微信扫码登录
> - websocket消息推送
